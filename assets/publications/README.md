# Publication figures

Put one representative figure for each selected paper in this folder. Prefer a clean method overview, framework diagram, or key result figure rather than a full paper page.

Use these filenames:

- `reasoning-taxonomy.png` — *Rethinking Reasoning: A Survey on Reasoning-based Backdoors in LLMs*
- `dup.png` — *DUP: Detection-guided Unlearning for Backdoor Purification in Language Models*
- `spbas.png` — *Syntactic Paraphrase-based Synthetic Data Generation for Backdoor Attacks against Chinese Language Models*
- `COT-TBA.png` — *CoT-TBA: Chain-of-Thought Truncation Backdoor Attacks against Large Reasoning Models*
- `transguard.png` — *TransGuard: Black-Box Defense Against Instruction Backdoors in LLM-Based Machine Translation*

Recommended format: PNG or JPG, landscape ratio around 16:9 to 4:3, at least 1200 px wide. Avoid screenshots with tiny unreadable text or large white page margins.
